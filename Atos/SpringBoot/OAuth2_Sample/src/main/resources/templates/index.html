OAuth2 Authorization Code Flow

A self-contained demo that acts as authorization server, OAuth2 client, and resource server on port 8080.

1. Start the flow

Sign in via the OAuth2 client. You will be redirected to the authorization server, authenticate, and approve consent.

Login with OAuth2

2. View tokens

After login, inspect the access token, refresh token, and OIDC claims on the dashboard.

Open dashboard

3. Call protected API

Use the access token as a Bearer token against /api/user and /api/hello.
