# Meeting Norms — Reduce Friction

## Defaults

- No meeting without agenda doc linked 24h ahead
- Default length: 25m (not 30m) or 50m (not 60m)
- Optional attendees leave if not needed in first 5m
- Decisions recorded in ticket or ADR — not chat-only

## Protected focus time

- Core hours 10:00–15:00: no recurring meetings without TL approval
- Incident/war room exempt

## Ceremonies (example team)

| Meeting | Cadence | Max | Purpose |
|---------|---------|-----|---------|
| Standup | Daily | 15m | Blockers only |
| Refinement | Weekly | 45m | Next sprint scope |
| Retro | Biweekly | 60m | Process improvement |
| Arch review | As needed | 45m | ADR-level decisions |

## Async first

- Design questions → architecture channel + ADR draft
- How-to questions → RAG / Stack Overflow for Teams before live call
- Status → board updates, not status meetings
