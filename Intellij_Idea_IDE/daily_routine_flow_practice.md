## 🔁 Daily Routine (no shortcuts — recall test)

---

### 1. 🔥 Warm-up (2 mins)

Open any file and **force yourself to use shortcuts only** (no mouse):

* Search for a class or file
* Navigate back and forward between locations
* Run the current class or test

👉 Goal: “navigation mindset”

---

### 2. ✍️ Editing drill (5 mins)

Create a small dummy class and repeat this flow **every day**:

1. Generate a constructor (or other boilerplate)
2. Trigger a quick fix / auto-import something
3. Comment and uncomment lines
4. Duplicate lines
5. Move lines up and down
6. Delete lines
7. Rename a variable

👉 Do this **same sequence daily** → builds muscle memory fast

---

### 3. 🧼 Cleanup habit (2 mins)

After editing anything:

* Optimize imports
* Reformat code

👉 Make this **automatic at the end of every change**
