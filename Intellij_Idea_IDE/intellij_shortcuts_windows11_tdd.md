## IntelliJ IDEA — Windows 11 Shortcuts for Java / Scala Development

Use the **Default** keymap in IntelliJ (`Settings` → `Keymap` → **Default**). On Windows 11, **Ctrl** and **Alt** replace macOS **⌘** and **⌥**; run/debug use **Shift+F10** / **Shift+F9** instead of **Ctrl+R** / **Ctrl+D**. If **Ctrl+Alt+Left** / **Ctrl+Alt+Right** rotate the display instead of navigating, disable those shortcuts in Windows display settings or rebind them in IntelliJ.

# 🔎 IDE FLOW
```
Shift Shift       → search everywhere
Ctrl+Alt+B        → go to implementation
Ctrl+Alt+Left     → navigate back
Ctrl+Alt+Right    → navigate forward
Alt+Enter         → quick fix / intention actions / static import
Ctrl+/            → toggle line comment
Ctrl+D            → duplicate line
Alt+Insert        → generate code (constructor / getter / setter)
Ctrl+Alt+O        → optimize imports
Ctrl+Alt+L        → reformat code
Shift+F10         → run current class/test

Alt+Enter         → select copilot
Alt+Shift+Up/Down → move line
Ctrl+Y            → delete line
Shift+F6          → rename variable
Ctrl+C            → highlight line
```

# 🔎 NAVIGATION

