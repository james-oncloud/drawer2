Here are **common IntelliJ IDEA code-generation shortcuts (Live Templates) on macOS** that developers use frequently during Java/Scala coding.

### Basic Code Templates

Type the abbreviation then press **`Tab`**.

```
sout      → System.out.println();
soutv     → System.out.println("var = " + var);
serr      → System.err.println();
psvm      → public static void main(String[] args)
main      → main method (alternative template)
```

### Loops

```
fori      → for (int i = 0; i < ; i++)
itar      → iterate array
itli      → iterate List
itco      → iterate collection
foreach   → enhanced for loop
```

### Conditionals

```
ifn       → if (var == null)
inn       → if (var != null)
inst      → if (obj instanceof Type)
```

### Method / Variable Helpers

```
var       → introduce variable
const     → introduce constant
return    → return value template
```

### Try/Catch

```
try       → try/catch block
tryf      → try/finally
tryc      → try/catch
```

### Common Java Templates

```
new       → create new object
cast      → type cast
```

### Useful Testing Templates

```
test      → test method template
assert    → assertion template
```

💡 **Tip:**
Press **`⌘ J`** inside the editor to see **all available live templates** for the current context.
