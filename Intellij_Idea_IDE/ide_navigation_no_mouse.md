Use the keyboard to **focus → navigate → open**:

---

# 📂 Method 1 (fastest)

* Open file search → `⌘ ⇧ O`
* Start typing file name
* Use `↑ / ↓` to highlight
* Press `Enter` to open

👉 This replaces clicking files entirely.

---

# 📁 Method 2 (project tree navigation)

* Focus Project panel → `⌘ 1`
* Use:

  * `↑ / ↓` → move between files
  * `→` → expand folder
  * `←` → collapse folder
* Press `Enter` → open selected file

---

# 🔍 Method 3 (jump to current file in tree)

* Reveal file in project → `⌘ ⌥ F1`, then press `1`
* Then navigate with arrows if needed

---

# ⚡ Method 4 (recent files)

* Open recent files → `⌘ E`
* Navigate with arrows
* Press `Enter`

---

# 🧠 Best practice

👉 Don’t rely on the project tree
👉 Use **search (`⌘ ⇧ O`) as your default**
