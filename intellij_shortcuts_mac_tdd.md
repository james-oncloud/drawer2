## IntelliJ IDEA — macOS Shortcuts for Java / Scala Development

### Quick Reference (TDD / Pair Programming)

---

# 🔎 NAVIGATION

```
⇧ ⇧     → search everywhere
⌘ B     → jump to declaration / definition
⌥ ⌘ B   → go to implementation
⌘ E     → recent files
⌘ ⇧ E   → recently edited files
⌘ ⌥ ←   → navigate back
⌘ ⌥ →   → navigate forward
⌘ F12   → file structure (methods / fields)
```

---

# 🧠 CODE UNDERSTANDING

```
⌘ P     → show method parameters
⌘ Q     → quick documentation
⌃ H     → type hierarchy
⌘ ⌥ H   → call hierarchy
```

---

# ⚡ QUICK FIX / INTENTIONS

```
⌥ ⏎     → quick fix / intention actions
```

---

# ✏️ EDITING

```
⌘ /     → toggle line comment
⌘ D     → duplicate line
⌘ Y     → delete line
⌥ ⌘ T   → surround with (if / try / etc)
```

---

# 🔧 REFACTORING (VERY COMMON IN TDD)

```
⇧ F6    → rename symbol
⌘ ⌥ V   → extract variable
⌘ ⌥ M   → extract method
⌘ N     → generate code (constructor / getter / setter)
```

---

# 🧹 CODE CLEANUP

```
⌥ ⌘ O   → optimize imports
⌥ ⌘ L   → reformat code
```

---

# ▶️ RUNNING / TESTING

```
⌃ R     → run current configuration
⌃ D     → debug current configuration
```

---

# 🧪 COMMON TDD FLOW (PAIR PROGRAMMING)

```
⌥ ⏎     → quick fix / import class
⌘ B     → jump to definition
⇧ F6    → rename
⌘ ⌥ M   → extract method
⌘ ⌥ V   → extract variable
⌥ ⌘ O   → optimize imports
⌥ ⌘ L   → format code
⌃ R     → run tests
```
