# Python Distilled - Study Outline

This is a study-oriented outline (not the book's official table of contents).  
It is organized as chapter titles with concise contents you can use for notes or practice.

## Chapter 1: Python Basics

**Contents**
- Running Python code (REPL, scripts)
- Variables, names, and object references
- Basic data types: numbers, strings, booleans, `None`
- Input/output and simple expressions
- Basic program structure and style

## Chapter 2: Control Flow

**Contents**
- `if`, `elif`, `else`
- `while` and `for` loops
- `break`, `continue`, `pass`
- Iteration patterns with `range()` and `enumerate()`
- Comprehensions overview

## Chapter 3: Functions

**Contents**
- Defining and calling functions
- Parameters, defaults, keyword arguments
- Variable-length args (`*args`, `**kwargs`)
- Return values and multiple returns
- Scope rules (`local`, `global`, `nonlocal`)

## Chapter 4: Strings and Text

**Contents**
- String creation and indexing/slicing
- Common string methods
- Formatting (`f-strings`, `format()`)
- Text processing patterns
- Unicode basics

## Chapter 5: Collections

**Contents**
- Lists, tuples, sets, dictionaries
- Mutability vs immutability
- Common collection operations
- Sorting and key functions
- Choosing the right collection type

## Chapter 6: Iteration and Generators

**Contents**
- Iterator protocol and `iter()/next()`
- Generator functions and `yield`
- Generator expressions
- `yield from`
- Lazy pipelines for data processing

## Chapter 7: Files and I/O

**Contents**
- Reading and writing text/binary files
- Context managers (`with`)
- File paths with `pathlib`
- Basic serialization (`json`, `csv`)
- Error-safe file handling patterns

## Chapter 8: Exceptions and Error Handling

**Contents**
- `try`, `except`, `else`, `finally`
- Raising exceptions
- Custom exception classes
- Exception chaining
- Defensive programming practices

## Chapter 9: Modules and Packages

**Contents**
- Imports and module search path
- Building reusable modules
- Package structure and `__init__.py`
- Running modules as scripts
- Basic dependency organization

## Chapter 10: Classes and OOP

**Contents**
- Defining classes and instances
- Instance/class attributes
- Methods, `self`, and initialization
- Inheritance and composition
- Special methods (`__repr__`, `__str__`, etc.)

## Chapter 11: Advanced Object Features

**Contents**
- Properties and descriptors (conceptual overview)
- Data model methods
- `dataclasses`
- Equality, ordering, and hashing
- API design for class-based code

## Chapter 12: Functional Tools

**Contents**
- Higher-order functions
- `lambda`, `map`, `filter`, `sorted`
- `functools` (`partial`, `lru_cache`, `reduce`)
- `itertools` building blocks
- Practical transformation pipelines

## Chapter 13: Testing and Debugging

**Contents**
- Writing unit tests with `unittest` or `pytest`
- Assertions and test organization
- Basic mocking concepts
- Debugging with `pdb` / `breakpoint()`
- Reproducible bug reports

## Chapter 14: Typing and Code Quality

**Contents**
- Type hints and annotations
- Common types from `typing`
- Static checking concepts (e.g., mypy)
- Linting and formatting workflows
- Readability and maintainability guidelines

## Chapter 15: Concurrency and Async

**Contents**
- Threading and multiprocessing overview
- `concurrent.futures`
- Async fundamentals (`async`/`await`)
- Basic `asyncio` patterns
- Choosing the right concurrency model

## Chapter 16: Practical Python Patterns

**Contents**
- Command-line tools (`argparse`)
- Logging best practices
- Configuration and environment variables
- Packaging basics and virtual environments
- Performance profiling and optimization checklist
