Cats is a functional programming library for Scala that provides type classes and data types for pure, composable, law-abiding code. It standardizes abstractions like Functor, Monad, Applicative, Semigroup, and Monoid, and offers data types such as Validated, EitherT, Kleisli, NonEmptyList, and Chain. It targets Scala 2 and Scala 3, and focuses on reusable, type-safe functional patterns with strong laws and instances for common Scala types.
